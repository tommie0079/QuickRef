# Notes
## 1. Install docker Ubuntu

[Setup](https://docs.docker.com/engine/install/ubuntu/)

## 2. Create directory
```
mkdir mc
```

 ## 3. Create file
```
sudo nano docker-compose.yaml
```

 ## 4. Compose
```
docker compose up -d 
```

 ## 5. Start the server
```
docker logs -f minecraft
```

 ## See what is happening
```
sudo docker ps
```
 ## Stop
```
sudo docker stop minecraft
```
